# Proyecto-AML
Proyecto de simulación y análisis de alertas transaccionales (AML) en R. Se generan datos sintéticos de clientes y transacciones, se aplican reglas de alerta por segmento y se construyen indicadores y visualizaciones descriptivas en un entorno reproducible con Quarto, pensado como insumo para dashboards en Power BI y portafolio profesional.
